**队列基础知识**
* 队列是连续的存储区，可以存储一系列的元素。是FIFO(先入先出，First- In-First-Out)结构。
* 队列通常具有头尾指针(左闭右开区间)，头指针指向第一个元素，尾指针指向最后一个元素的下一位。
* 队列支持(从队尾)入队(enqueue)、(从队首)出队(dequeue)操作。

**队列应用场景**
* CPU的超线程技术
* 线程迟的任务队列
